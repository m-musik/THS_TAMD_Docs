To apply TAM or TAMD to your stem buses, you will want to load an instance of YSFX as **the first** insert on each stem bus. Then, load the JSFX file for TAM or TAMD. By default, it will look like the following:

![Screenshot](../img/Pasted%20image%2020260601223300.png)

All stem buses need to be configured as a **Bus**. To make this change, click the **CHANNEL** button near the top of the plugin GUI, then click **Bus**. This will update the topology of the instance to a bus, and the plugin will now appear like the following:

![Screenshot](../img/Pasted%20image%2020260601223411.png)

Repeat this process for all stem buses in your project, making sure to set each instance to a **Bus** rather than a **Channel**.
