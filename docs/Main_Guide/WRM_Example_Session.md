Below is an example session in Cubase Pro 15 containing 4 stem busses of 3 individual tracks each, all connected together using the Wrapper and Router method. In the example, I am using Metaplugin as a wrapper and Blue Cat's Connector as a router. To demonstrate the routing, I have added TestGenerator instances before YSFX on all individual tracks to act as an input signal to pass through TAM or TAMD and THS. When fully configured, the session appears as follows in Cubase's MixConsole view:

![Screenshot](../img/Pasted%20image%2020260601225042.png)

Note that this is a barebones session meant to illustrate the base connectivity needed to operate THS and TAM or TAMD in a project.
