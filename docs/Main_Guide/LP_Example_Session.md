Below is an example session in Logic Pro 12.2 containing 4 stem busses of 3 individual tracks each, all connected together using the Metaplugin and Connector method. To demonstrate the routing, I have added various stock Apple loops to each individual track to act as an input signal to pass through TAM or TAMD and THS. When fully configured, the session appears as follows in Logic's mixer

![Screenshot](../img/Pasted%20image%2020260606212920.png)

Note that this is a barebones session meant to illustrate the base connectivity needed to operate THS and TAM or TAMD in a Logic Pro project.
