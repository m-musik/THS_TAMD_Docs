Below is an example session in Cubase containing 4 stem buses of 3 individual tracks each, all connected together using the Metaplugin and Connector method. To demonstrate the routing, I have added TestGenerator instances before YSFX on all individual tracks to act as an input signal to pass through TAM or TAMD and THS.
